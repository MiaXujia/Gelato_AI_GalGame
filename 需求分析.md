1. 需求报告 (Requirements Report)
项目名称： AutoGal - AI 驱动的 Ren'Py 游戏生成 Agent

项目目标：
创建一个自动化的 Python AI Agent，用户只需输入一段自然语言的“场景描述”或“故事大纲”，Agent 即可自动生成完整的剧本、角色立绘、背景图像，并最终将其打包成一个可直接运行的 Ren'Py 游戏工程。

核心功能需求：

输入解析： 接收并理解用户提供的提示词（Prompt）。

剧本生成引擎： 调用 LLM（如 GPT-4）生成符合 Ren'Py 语法的结构化剧本（包含定义、对话、场景切换、分支选项）。

资产生成引擎： * 根据剧本自动提取绘图提示词。

调用图像生成 API（如 DALL-E 3 或 Stable Diffusion）生成背景 (BG) 和立绘 (CG/Sprite)。

项目组装器： * 自动复制 Ren'Py 基础空模板。

将生成的图片放入 game/images/ 目录。

将生成的剧本写入 game/script.rpy 文件。

非功能需求：

模块化： LLM 对接、图像生成、文件操作应相互独立，便于未来替换不同的模型 API。

容错处理： 确保 LLM 输出的代码格式正确，对异常生成的图片下载失败有重试机制。
